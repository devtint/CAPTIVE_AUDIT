Captive Portal Security Audit

Run security audits on your own captive portals directly from your Android device using Termux. Analyze results with the built-in JSON viewer.

Get Started Analyze Report @BadCodeWriter

Step-by-Step Guide

Run security audits on your own captive portal from Termux on Android.

Disclaimer: Only run this on portals you OWN or have written permission to test. Unauthorized use is illegal.
1

Install Termux

Download Termux from F-Droid (recommended) — not Google Play.

https://f-droid.org/en/packages/com.termux/
2

Update Packages

Open Termux and update all system packages first.

pkg update -y && pkg upgrade -y
3

Install Python

Install Python and pip package manager.

pkg install python -y
4

Install Dependencies

Install the required Python packages.

pip install requests urllib3
5

Download the Scripts

Option A: Clone with git (recommended)

pkg install git -y git clone https://github.com/devtint/CAPTIVE_AUDIT.git cd CAPTIVE_AUDIT

Option B: Download directly (no git needed)

mkdir CAPTIVE_AUDIT && cd CAPTIVE_AUDIT curl -LO https://raw.githubusercontent.com/devtint/CAPTIVE_AUDIT/main/captive_portal_audit.py curl -LO https://raw.githubusercontent.com/devtint/CAPTIVE_AUDIT/main/starlink.py

If curl is not installed: pkg install curl -y

6

Connect to Captive Portal WiFi

Go to your Android WiFi settings and connect to the captive portal network you want to audit. Do NOT accept the portal login yet.

7

Run the Audit Script

Execute the captive portal security audit tool.

python captive_portal_audit.py
8

Follow the Interactive Menu

The script will guide you through:

  • Select your language (English / Myanmar)
  • Enter the portal URL (e.g. http://192.168.1.1)
  • Optionally enter gateway host/IP and port
  • The audit runs 7 security checks automatically
9

Get the JSON Report

After the audit, a JSON report file is saved. You can view or upload it to the Analyzer below.

cat captive_portal_audit_report.json

To copy the file to your phone storage:

cp captive_portal_audit_report.json /sdcard/Download/
1

Complete Steps 1–5 from Security Audit

Make sure Termux, Python, dependencies, and the repository are already set up.

2

Connect to Captive Portal WiFi

Connect to the portal WiFi network. The Turbo Engine will auto-detect the captive portal.

3

Run the Turbo Engine

Launch the Turbo Network Engine script.

cd CAPTIVE_AUDIT python starlink.py
4

Select Option 1 from Menu

Choose [1] Starlink Hack to start the Turbo Network Engine.

5

How It Works

The engine will automatically:

  • Detect the captive portal redirect
  • Capture the session ID from the portal
  • Probe the voucher API endpoint
  • Launch multi-threaded auth pings to the gateway
  • Monitor internet connectivity in real-time
6

Stop the Engine

Press Ctrl + C to stop and return to the menu.
